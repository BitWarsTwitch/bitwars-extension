Setup Guide

1. Connect to BitWars from OBS
Add your custom URL to OBS on a Browser Source. If you don't know how, look up "OBS Browser source" online.
2. (Optional) Add a Co-Streamer Code
In order to create a play session with another streamer, each of you have to put eachother's friend code in the following field. Leave empty if you want to use the plugin alone.
3. Have Fun & Share
Thanks for using BitWars. This has been created as part of a Twitch Hackathon. Sharing helps us grow and improve our extension!
