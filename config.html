Setup Guide

1. Configure your Profile
Setup the name of your tower! This will appear in the left side of your screen.
2. Choose Your Game Mode
In order to create a play session with another streamer, each of you have to put eachother's friend code in the following field.
Play alone without adding a friend code. For example, you can have each tower be an option users can 'fight' for. Set the name of the second tower below.
3. Connect to BitWars from OBS
Add your custom URL to OBS on a Browser Source. If you don't know how, look up "OBS Browser source" online.
